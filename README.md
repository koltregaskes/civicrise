# Civicrise

Modern city-building game project for E-lusion Studios.

This repo is the clean standalone home for `Civicrise`. The older `isometric-city` fork is now treated as a reference-only source, not the long-term product repo.

## Current status

- new standalone repo created
- production direction defined
- dedicated implementation workspace ready
- next step is building the first reviewable city-builder demo

## Product direction

`Civicrise` is being developed as:

- `Phase 1`: browser-first visible demo
- `Phase 2`: desktop production version

The first target is one strong city-builder district with zoning, traffic, budget pressure, and a readable civic simulation loop.
